# UNO Game Backend

## Installation

```bash
git clone https://github.com/Comkeen1984/uno-game-backend.git
cd uno-game-backend
npm install
```

## Lokaler Start

```bash
npm run dev    # startet server.js mit nodemon
# oder
npm start      # startet server.js normal
```

Der Server läuft dann auf http://localhost:10000.
